# llama.cpp and Open WebUI
Running [llama.cpp](https://github.com/ggml-org/llama.cpp) and [Open WebUI](https://github.com/open-webui/open-webui) on GPU AMD Radeon.

### Launch
```bash
git clone https://github.com/HardAndHeavy/llama-open-webui
cd llama-open-webui
make run
```

Open http://localhost.
